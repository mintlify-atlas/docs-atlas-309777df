### Validation Rules

The system validates the following business rules:

```java
// From GastoService.java:73-83
private void validarDatosGasto(String titulo, LocalDate fecha, BigDecimal monto) {
    if (titulo == null || titulo.trim().isEmpty()) {
        throw new GastoException("El título del gasto no puede estar vacío");
    }
    if (fecha == null) {
        throw new GastoException("La fecha del gasto es obligatoria");
    }
    if (monto == null || monto.compareTo(BigDecimal.ZERO) <= 0) {
        throw new GastoException("El monto del gasto debe ser mayor a cero");
    }
}
```

<Warning>
**Validation Errors:**
- Title cannot be empty or contain only whitespace
- Date is required and must be in `YYYY-MM-DD` format
- Amount must be greater than zero
</Warning>

### ID Matching Logic

```java
// From GastoService.java:86-104
private UUID obtenerUUIDPorIdCorto(String idCorto) {
    if (idCorto == null || idCorto.length() < 8) {
        throw new GastoException("Debe ingresar al menos los primeros 8 caracteres del ID");
    }

    List<UUID> coincidencias = repository.findAll().stream()
            .map(Gasto::getId)
            .filter(id -> id.toString().startsWith(idCorto))
            .toList();

    if (coincidencias.isEmpty()) {
        throw new GastoException("No se encontró ningún gasto que coincida con el ID proporcionado");
    }
    if (coincidencias.size() > 1) {
        throw new GastoException("El ID ingresado coincide con múltiples gastos. Ingrese más caracteres");
    }

    return coincidencias.get(0);
}
```

<Warning>
**Common Errors:**
- `Debe ingresar al menos los primeros 8 caracteres del ID` - ID too short
- `No se encontró ningún gasto que coincida con el ID proporcionado` - ID not found
- `El ID ingresado coincide con múltiples gastos` - Ambiguous ID, need more characters
</Warning>

### Implementation Details

```java
// From Consola.java:104-124
private static void eliminarGasto() {
    System.out.println("\n--- ELIMINAR GASTO ---");
    System.out.print("Ingrese el ID (8 caracteres) del gasto (o '0' para cancelar): ");
    String idCorto = scanner.nextLine().trim();

    if (idCorto.equals("0")) {
        System.out.println("Operación cancelada por el usuario");
        return;
    }

    System.out.print("¿Está seguro que desea eliminar este gasto? (S/N): ");
    String confirmacion = scanner.nextLine().trim().toUpperCase();

    if (!confirmacion.equals("S")) {
        System.out.println("Eliminación cancelada");
        return;
    }

    servicio.eliminarGasto(idCorto);
    System.out.println("Gasto eliminado exitosamente");
}
```

### Safety Features

<Accordion title="Two-Step Cancellation">
  You can cancel the deletion in two ways:
  1. Enter `0` when prompted for the ID
  2. Enter anything other than `S` when asked for confirmation
</Accordion>

<Accordion title="Validation Before Deletion">
  ```java
  // From GastoService.java:45-53
  public void eliminarGasto(String idCorto) {
      UUID idReal = obtenerUUIDPorIdCorto(idCorto);

      repository.findById(idReal)
              .orElseThrow(() -> new GastoException(
                      "No se puede eliminar porque no se encontró un gasto con el ID: " + idReal));

      repository.deleteById(idReal);
  }
  ```
  The system verifies the expense exists before attempting deletion.
</Accordion>

```java
// From Consola.java:137-156
private static void filtrarGastos() {
    System.out.println("\n--- FILTRAR GASTOS POR MES ---");
    try {
        System.out.print("Ingrese el mes (1-12): ");
        int mes = Integer.parseInt(scanner.nextLine());

        System.out.print("Ingrese el año (ejemplo: 2026): ");
        int anio = Integer.parseInt(scanner.nextLine());

        List<Gasto> filtrados = servicio.filtrarGastosPorMes(mes, anio);

        if (filtrados.isEmpty()) {
            System.out.println("No se encontraron gastos en esa fecha");
        } else {
            filtrados.forEach(System.out::println);
        }
    } catch (NumberFormatException e) {
        System.out.println("Error. Ingrese valores numéricos para el mes y el año");
    }
}
```

### Filter Logic

```java
// From GastoService.java:61-70
public List<Gasto> filtrarGastosPorMes(int mes, int anio) {
    if (mes < 1 || mes > 12) {
        throw new GastoException("El mes debe estar entre 1 y 12");
    }

    return repository.findAll().stream()
            .filter(gasto -> gasto.getFecha().getMonthValue() == mes && gasto.getFecha().getYear() == anio)
            .sorted(Comparator.comparing(Gasto::getFecha).reversed())
            .toList();
}
```

<Warning>
**Validation:**
- Month must be between 1 and 12
- Both month and year must be valid integers
- Results are sorted by date (newest first)
</Warning>

## Input Validation Helpers

The application includes helper methods to ensure valid data entry:

### Date Validation

```java
// From Consola.java:159-168
private static LocalDate leerFecha() {
    while (true) {
        System.out.print("Fecha (Formato AAAA-MM-DD): ");
        try {
            return LocalDate.parse(scanner.nextLine());
        } catch (DateTimeParseException e) {
            System.out.println("Formato de fecha incorrecto, use AAAA-MM-DD (ejemplo: 2026-03-02)");
        }
    }
}
```

<Note>
The date input loops until a valid format is provided. Use format: `YYYY-MM-DD`
</Note>

### Amount Validation

```java
// From Consola.java:170-179
private static BigDecimal leerMonto() {

    }
```

<Note>
